Git adalah sebuah sistem pengontrol versi (Version Control System) yang digunakan untuk mengelola perubahan dalam setiap file dalam suatu proyek, biasanya digunakan dalam pengembangan software. 
Git memiliki beberapa fitur yang memudahkan pengembangan kolaboratif, seperti merge, branching, dan staging area. Sistem ini dikembangkan oleh Linus Torvalds pada tahun 2005 dan sejak itu telah menjadi salah satu sistem pengontrol versi terpopuler di dunia. Git dapat digunakan secara offline atau online, dan memiliki beberapa implementasi yang dapat digunakan di berbagai platform, seperti C, Java, Go, dan Python.
